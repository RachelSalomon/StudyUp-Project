# StudyUp - Student Task Manager

StudyUp is a smart, full-stack task and course management web application tailored specifically for students to organize their academic life.

## Tech Stack
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB (Mongoose)
- **Authentication:** JSON Web Tokens (JWT)
